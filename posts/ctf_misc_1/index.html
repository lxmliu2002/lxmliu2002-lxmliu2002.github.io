文件操作与隐写#

文件类型识别#

File 命令#

当文件无后缀名无法打开时，使用 file 命令可识别出文件类型。

1

file <filename>

Winhex#

十六进制文件编辑器

通过 winhex 程序可查看文件头类型，以此判断文件类型。

文件头残缺、错误#

文件无法正常打开包括文件头部残缺和文件头部错误两种情况。针对残缺情况，使用 winhex 添加相应的文件头，针对错误情况，找一个同类型的文件进行替换。

1

file <filename>

可以看到其显示为 data，表示其为数据文件，使用二进制文件编辑器为其添加文件头即可。

文件分离操作#

Binwalk#

Binwalk 为 Linux 下分析和分离文件的工具，用于分辨文件是否是多个文件合并而成，并进行分离，在同目录下生成一个 _filename_extracted 文件目录，其中有分离后的文件。

1
2

分析文件：binwalk <filename>
分离文件：binwalk -e <filename>

foremost#

若 binwalk 未能正确分离文件，使用 foremost，成功执行后，会在目标文件的文件目录下生成给定的目录，目录中按文件类型分离出文件。

1

foremost <filename> -o <contentsname>

dd#

dd 手动分离文件。

1
2
3
4
5

dd if=<filename> of=<output_filename> bs=1 skip=<begin_blocks>
# if = file 输入文件名，缺省为标准输入
# of = file 输出文件名，缺省为标准输出
# bs = bytes 同时设置读写块的大小为 bytes，可代替 ibs 和 obs
# skip = blocks 从输入文件开头跳过 blocks 个块后再开始复制

winhex#

将目标文件拖入 winhex，找到要分离的部分，复制出来即可。

010Editor4#

同 winhex。

文件合并操作#

Linux#

1

cat <input_file1> <input_file2> ... <input_filen> > <output_file>

完整性检测：

1

md5sum <filename>

Windows#

1

copy /B <input_file1>+<input_file2>+ ... +<input_filen> <output_filename>

完整性检测：

1

certutil -hashfile <filename> md5

Firework#

使用 winhex 打开文件时看到文件头中的 firework 标识，则可以通过 firework 查看隐藏图片。

Exif#

Exif 按照 JPEG 的规格在 JPEG 中插入一些图像或数字相机的信息数据以及缩略图像，可以通过与 JPEG 兼容的软件来查看 Exif 格式的图像文件。

Stegsolve#

当两张图片外观、大小、像素等基本相同时，可以使用 Stegsolve 将两张图片进行 ADD、SUB、异或等操作。

LSB（最低有效位 Least Significant Bit）#

LSB 替换隐写基本思想是用嵌入的秘密信息取代载体图像的最低比特位，原来的 7 个高位平面与替代秘密信息的最低为平面组合成含隐藏信息的新图形。

  1. 像素三原色 RGB

  2. 通过修改像素中最低为的 1 bit 来达到隐藏效果

  3. 工具

    • stegsolve

    • zsteg

      1

      zsteg <filename>

    • wbstego4

      • 解密 .bmp/pdf 格式文件

    • python 脚本

TweakPNG#

文件头正常却无法打开文件，可以利用 TweakPNG 修改 CRC。

CRC 没错，但是图片的高度或宽度发生错误，则需通过 CRC 计算出正确高度或宽度。

Bftools#

在 windows 的 cmd 下对加密过的图片文件进行解密。

1
2

Bftools.exe decode braincopter <filename> -output <outputname>
Bftools.exe run <outputname>

SilentEye#

windows 下对加密的图片进行解密。

JPG 图像加密#

Stegdetect 工具探测加密方式#

Stegdetect 主要分析 JPEG 文件，可以检测到通过 JSteg、JPHide、OutGuess、Invisible Secrets、F5、appendX 和 Camouflage 等工具的隐藏信息。

1

stegdetect -s <敏感度> <filename>

Jphide#

Jphide 基于最低有效位 LSB 的 JPEG 格式图像隐写算法。

Outguess#

Stegdetect 识别出来或是题目提示是 outguess 加密的图片。

1
2

编译：./configure && make && make install
outguess -r <filename> <outputname>

F5#

Stegdetect 识别出来或是题目提示是 F5 加密的图片。

1

Java Extact <filename> -p <crypto>

二维码处理#

使用 CQR.exe 打开图片查看其内容。

若二维码定位角被覆盖导致无法识别，则使用 PS 或画图工具将其补齐.

如果其定位点中间为白色，可能被反色，使用画图工具将颜色矫正再扫描即可。
