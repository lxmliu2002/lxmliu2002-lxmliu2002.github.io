教育基础知识#

一、教育的产生与发展（重点）#

（一）教育的含义#

1. “教育”一词的由来（单选）#

“教育”一词最早见于**《孟子·尽心上》“**得天下英才而教育之，三乐也”

2. 教育的概念（单选 + 辨析）#
  • 广义：泛指一切增进人的知识和技能、发展人的智力和体力、影响人的思想和品德的活动。它包括社会教育、学校教育和家庭教育（三合力）。
  • 狭义：学校教育是教育者根据一定的社会要求，有目的、有计划、有组织地对受教育者施加影响，把他们培养成一定社会所需要的人的活动。

（二）教育的基本要素（单选）#

  • 教育者（包括起主体地位的学校教师、管理人员等）：主导作用
  • 教育影响：教育内容、教育手段，纽带、中介
  • 受教育者：学习的主体

（三）教育的属性（重点：单选 + 辨析）#

（一）人的发展概述（了解）#

（二）个体身心发展的一般规律（重点）#

  1. 顺序性

    • 表现：由低级到高级，由简单到复杂，由量变到质变

      顺序不可跨越，方向不可逆

    • 启示：循序渐进，拔苗助长

    • 例子：欲速则不达；盈科而后进；不陵节而施；杂施而不逊，则坏乱而不修

  2. 阶段性

    • 表现：不同阶段有不同特征，不同矛盾，不同任务
    • 启示：分阶段教学，有针对性教学
    • 例子：幼儿园小学化；一刀切，一锅煮

  3. 不平衡性（个体内）

    • 表现：速度有快有慢，同一方面不同速，不同方面不同步
    • 启示：关键期——劳伦兹
    • 例子：狼孩；当其可之谓时；时过然后学，虽勤苦而难成

  4. 互补性

    • 表现：身身互补，身心互补
    • 启示：扬长避短，长善救失
    • 例子：盲人听力好，身残志坚

  5. 个体差异性（个体间）

    • 表现：群体之间，个体之间
    • 启示：因材施教
    • 例子：一把钥匙开一把锁；人心不同，各如其面

例：简述个体身心发展的一般规律。

  1. 个体身心发展具有顺序性。它是一个由低级到高级、由简单到复杂的连续不断的发展过程。
  2. 个体身心发展具有阶段性。不同的年龄阶段面临着不同的发展任务。
  3. 个体身心发展具有不平衡性。个体的发展并不总是按相同的速度直线前进的。
  4. 个体身心发展具有互补性。这种互补既体现在身体机能和身体机能之间，也体现在身体机能和心理机能之间。
  5. 个体身心发展具有个别差异性。这种差异体现在群体和群体之间以及个体和个体之间。

（四）影响人身心发展的因素（重点：单选 + 辨析 + 简答）#

1. 遗传（单选）#
  • 遗传素质是人的身心发展的生理前提，为人的身心发展提供了可能性。
  • 遗传素质的成熟程度制约着人的发展过程及年龄特征。
  • 遗传素质的差异性对人的发展有一定的影响。
  • 遗传素质对人的发展的影响大小与本身是否符合常态有关。
  • 遗传素质具有可塑性。
2. 环境（单选）#
  • 环境为个体的发展提供了多种可能，包括机遇、条件和对象。
  • 环境是人的身心发展的外部的客观条件，环境使遗传提供的发展可能性变成现实。
  • 环境不决定人的发展。人在接受环境影响时不是消极的、被动的。人具有主观能动性。

一、课程概述（重点）#

（一）课程概述#

1. “课程”一词的由来（单选）#

  • 唐朝：孔颖达“维护课程，必君子监之，乃依法制”

  • 朱熹：“宽着期限，紧着课程”

    朱熹最早把“课程”一词有意义的现实关联

  • 斯宾塞：西方最早提出“课程”一词专门术语

  • 博比特：专门领域，1928年出版第一本课程论专著——《课程》

2. “课程”的概念 （了解）#
  • 广义的课程：广义的课程是指为实现学校教育目的而选择的教育内容的总和及进程安排。它包括各门学科和课外活动。
  • 狭义的课程：狭义的课程是指某一门学科。如语文、数学等。

（二）课程的类型（重点：单选）#

1. 按照内容属性划分#
  • 学科课程
    • 知识的逻辑体系
    • 间接经验（别人授予）
  • 活动（ 经验） 课程
    • 学生的兴趣和经验
    • 直接经验（亲身经历）
    • 特点：逻辑性、系统性、简约性
2. 按照组织方式划分#
  • 分科课程（同学科课程）
  • 综合课程
    • 学科整合
    • 主张把两门或两门以上学科的内容合并起来组成课程
3. 按照实施要求划分#
  • 必修课程：培养学生的共性
  • 选修课程：发展学生的个性
4. 按照设计、开发和管理主体划分#
  • 国家课程：体现国家的意志
  • 地方课程：满足社会发展的需要
  • 校本课程：展示学校的特色
    • 针对性，过程性，民主性，多元性（主体的多元性）
5. 按照课程任务划分#
  • 基础型课程：培养学生的基础能力（读、写、算等）
  • 拓展型课程：拓宽学生的视野（广度）
  • 研究型课程：培养学生的探究态度和能力（自己探究，创新）
6. 按照呈现方式划分#
  • 显性课程：以直接的、明显的方式呈现，具有计划性（课表上给出）
  • 隐性课程：以间接的、内隐的方式呈现的课程（计划外，如校风、班风、校报等）

（三）课程理论流派#

  • 学科中心课程：赫尔巴特
    • 以学科逻辑体系为核心
    • 主张分科教学
  • 活动中心课程：杜威
    • 以儿童活动为中心
    • 课程组织应心理学化
    • 儿童的需要和兴趣
  • 社会中心课程：布拉梅尔德
    • 社会问题为中心

三、教学原则和方法#

（一）教学原则（重点：全题型）#

1. 直观性原则（重点）#
  • 概念：清晰表象、感性认识
  • 古语：
    • 荀子：不闻不若闻之，闻之不若见之
    • 夸美纽斯：一切的知识都是从感官的感知开始的
  • 分类：实物直观、模象直观、言语直观
  • 贯彻要求
    • 正确选择直观教具和现代化教学手段
    • 直观与讲解结合
    • 重视运用语言直观
    • 要防止直观不当与滥用
2. 启发性原则（重点）#
  • 概念：调动主动性、积极性、引导其思考
  • 古语：
    • 孔子：不愤不启，不悱不发
    • 苏格拉底：产婆术
    • 《学记》：道而弗牵，强而弗抑，开而弗达
  • 运用要求：
    • 加强学习的目的性教育，调动学生学习的主动性
    • 启发学生独立思考，发展学生的逻辑思维能力
    • 让学生自己动手，培养学生独立解决问题的能力
    • 发扬教学民主
3. 循序渐进原则（重点）#
  • 概念：按知识逻辑顺序 + 学生认识发展的顺序教学
  • 古语：
    • 《学记》：学不躐等；不陵节而施
    • 孟子：盈科而后进
    • 朱熹：未得乎前，则不敢求其后
  • 贯彻要求：
    • 教学要有系统性
    • 注意主要矛盾，解决好重点与难点的教学
    • 按照学生的认识顺序，由浅入深、由易到难进行教学
4. 巩固性原则#
  • 概念：长久的保持
  • 古语：
    • 孔子：学而时习之；温故而知新
    • 乌申斯基：复习是学习之母
  • 贯彻要求：组织好学生的复习工作，教会学生记忆方法
5. 量力性原则#
  • 概念：能够接受，又有一定难度
  • 古语：
    • 墨子：夫智者必量其力所能至而从事焉；
    • 《学记》：语之而不知，虽舍之可也。
6. 思想性与科学性相统一原则（重点）#
  • 概念：品德 + 知识
  • 古语：教育性教学——赫尔巴特
7. 理论联系实际原则#

知行合一

  • 概念：学懂学会、学以致用
  • 古语：

  • 贯彻要求：

（二）教学方法（重点）#

1. 教学方法的概念（了解）#

教学方法是为了完成教学任务而采用的办法。它包括教师教的方法和学生学的方法，是教师引导学生掌握知识技能、获得身心发展的方法。

2. 教学方法（重点：单选 + 材料分析 + 简答）#
  • 语言传递：讲授法、谈话法、讨论法、读书指导法
    • 讲授法
      • 定义：教师口头语言
      • 地位：运用最多、最广的教学方法
      • 分类：讲述、讲解、讲读、讲演
      • 要求：
        • 讲授内容要有科学性、系统性、思想性
        • 注意启发
        • 讲究语言艺术
        • 要组织学生听讲
    • 谈话法
      • 定义：教师提问、学生回答
      • 分类：复习谈话和启发谈话
    • 讨论法
      • 定义：学生探讨、辩论
      • 名言：相观而善；独学而无友，则孤陋而寡闻
      • 优点：集思广益
      • 缺点：需具备一定基础，多在高年级使用
    • 读书指导法
      • 定义：培养学生自学能力
      • 教师不指定书目
  • 直观感知：演示法、参观法
    • 演示法
      • 定义：**教师展示（老师做学生看）**实物、直观教具，进行示范性的实验、采取现代化视听手段
      • 目的：指导学生获得知识
    • 参观法
      • 定义：对实物进行实地观察、研究
      • 分类：准备（教学前）、并行（教学中）、总结性（教学后）参观
  • 实际训练：练习法、实验法、实习作业法、实践活动法
    • 练习法
      • 定义：学生运用所学知识独立进行操作
      • 目的：巩固知识
    • 实验法
      • 定义：学生使用仪器和设备，引起事物和现象产生变化
      • 目的：获得知识与技能
    • 实习作业法
      • 概念：学生运用所学知识在校内外一定场所进行实际操作
      • 目的：学以致用
    • 实践活动法
      • 概念：学生参加社会实践活动，培养解决实际问题的能力和多方面实践能力
      • 特点：从做中学
  • 情感陶冶：欣赏教学法、情景教学法
    • 欣赏教学法：真善美
    • 情景教学法：创设情境，情感体验

五、教学评价#

（一）教学评价的概念#

教学评价是指以教学目标为依据，通过一定的标准和手段，对教学活动及其结果给予价值上的判断，即对教学活动及其结果进行测量、分析和评定的过程。

（二）教学评价的分类（重点）#

  • 实施功能

    • 诊断性评价
      • 含义：学期教学开始或单元教学开始时所进行的评价
      • 例子：摸底考试、分班考试、入学考试
    • 形成性评价
      • 含义：在教学进程中进行的经常性评价
      • 例子：口头提问、随堂测验、档案袋评价
    • 总结性评价
      • 含义：在一个大的学习阶段结束后进行的全面评价
      • 例子：期末考试、毕业会考、结业考试

  • 运用标准

    • 相对性评价
      • 含义：又称为常模参照性评价，与别人比
      • 例子：高考、公务员考试、教师招聘考试
    • 绝对性评价
      • 含义：又称目标参照性评价，与目标比
      • 例子：会考、教师资格证考试、驾照考试
    • 个体内差异评价
      • 含义：对个体的过去和现在，或者不同方面进行比较，与自己比
      • 例子：偏科

  • 评价主体

    • 外部评价：被评价者之外的专业人员进行的评价
    • 内部评价：即自我评价

  • 评价方法

    • 定性评价：做**“质”**的分析
    • 定量评价：做**“量”**的分析

（三）知觉#

1. 知觉的概念（单选）#

2. 知觉的种类（单选）#
  • 空间知觉：形状、大小、方位、深度——视崖实验
  • 时间知觉
  • 运动知觉：真动知觉，似动知觉

4. 观察#
a. 观察的含义（了解）#

观察是人的一种有目的、有计划、持久的知觉活动，是知觉的高级形式，又叫“思维的知觉”。

b. 观察力的培养（简答）#
  • 引导学生明确观察的目的与任务，这是良好观察的重要条件。

（四）记忆（重点）#

1. 记忆的概念（单选）#

记忆是在头脑中积累和保存个体经验的心理过程，是人脑对外界输入的信息进行编码、储存和提取的过程。

2. 记忆的主要类型#

  1. 根据记忆内容和经验对象划分（单选）

    1. 形象记忆：感知过事物的具体形象的记忆
    2. 逻辑记忆：概念、公式、原理、规律等
    3. 情绪记忆：情绪、情感
    4. 动作记忆：运动状态和动作技能

  2. 根据记忆内容保持时间长短划分（单选 + 辨析）

    记忆类型 保持时间 容量
    瞬时记忆（感觉记忆） 0.25~1 秒 容量较大
    短时记忆（工作记忆） 1 分钟左右 7±2 个组块
    长时记忆 1 分钟以上 容量很大 / 无限

    瞬时记忆 –注意–> 短时记忆 –复述–> 长时记忆

  3. 信息加工和储存内容（单选）

    1. 陈述性记忆：静态
    2. 程序性记忆：怎么做

  4. 记忆过程意识参与程度（单选）

    1. 外显记忆：知道自己知道
    2. 内隐记忆：无意识中掌握
3. 记忆过程#
  1. 识记：开端，是人们获得个体经验的过程
  2. 保持：第二个环节，知识经验在人脑的巩固过程
  3. 回忆或再认：回忆不在眼前，再认在眼前
4. 遗忘#
a. 原因（单选 + 简答）#

  1. 痕迹衰退说——不复习

  2. 干扰抑制说——相互干扰

    1. 前摄抑制：前→后，干扰

    2. 倒摄抑制：后→前，干扰

      早上的学习不受前摄抑制 晚上的学习不受倒摄抑制 中午的学习受到前摄抑制、倒摄抑制

  3. 动机说——情绪、压抑

  4. 提取失败说——缺线索

  5. 同化说 ——认知结构简化

例：简述遗忘的原因。

  • 痕迹衰退说（自动消退）：认为遗忘是记忆痕迹得不到强化而逐渐衰退，以至于最后消退的结果。
  • 干扰抑制说：认为遗忘是在学习和回忆之间受到其他刺激的干扰所致。
  • 动机说（压抑说）：认为遗忘是由情绪或动机的压抑作用引起的。
  • 提取失败说：认为长时记忆中的信息无法回忆，是因为个体在提取时没有找到适当的提取线索。
  • 同化说：认为遗忘实际上是知识的组织与认知结构简化的过程，即用高级的概念与规律代替低级的观念，使低级观念发生遗忘，从而减轻记忆负担。
b. 规律（单选）#
  • 遗忘是学习之后立即发生的
  • 进程不均匀，先快后慢
  • 遗忘到一定程度，后期不遗忘
c. 影响因素（简答）#
  • 学习材料的性质；
  • 识记材料的数量和学习程度的大小；
  • 记忆任务的长久性与重要性；
  • 识记的方法；
  • 时间因素；
  • 情绪和动机。
5. 中学生记忆发展的特点（简答）#
  • 记忆容量逐渐增大，短时记忆的广度接近成人
  • 记忆的有意性加强
  • 意义识记成为主要记忆手段
  • 抽象记忆逐渐占据主导地位
6. 提高记忆能力，促进知识巩固#
a. 良好记忆品质的特点（单选）#
  • 敏捷性：速度快慢
  • 持久性：时间长短
  • 准确性：正确精确
  • 准备性：综合体现，提取
b. 提高记忆能力的方法（简答）#
  • 明确记忆目的，增强学习的主动性;
  • 理解材料的意义，少用机械记忆；
  • 对材料进行精细加工，促进深度理解;
  • 运用组块化学习策略，合理组织材料;
  • 运用多重编码方式，提高信息加工处理的质量;
  • 注重复习方法，防止知识遗忘。
c. 注重复习方法，防止遗忘（单选 + 材料分析）#

  1. 复习的时间。

    • 及时复习。
    • 合理分配时间。

  2. 复习的方法。

    • 做到分散复习与集中复习相结合。
    • 反复阅读与试图回忆相结合。
    • 复习方法要多样化。
    • 运用多种感官进行复习。

  3. 复习的次数。

    • 复习内容的数量要适当，一次复习内容的数量不宜过多。

    • 适当过度学习，过度学习 50%，即学习的熟练程度达到 150% 的学习。

      恰能背诵（100%）+ 过度学习（50%）= 学习熟练程度（150%）

三、学习理论#

（一）行为主义学习理论（重点）#

    • 主要规律
      • 获得与消退
        • 获得：建立条件反射的过程
        • 消退：条件反射消失
      • 刺激的泛化与分化
        • 泛化：类似的刺激也能诱发其条件反应
        • 分化：只对条件刺激有反应

  • 猫——桑代克 - 联结主义学习理论

    • 基本规律
      • 准备律（预习）
      • 练习律（做题）
      • 效果律（反馈）（最重要）

  • 鼠——斯金纳的操作性条件作用理论

    • 结论：学习实质上是一种反应概率上的变化，而强化是增强反应概率的手段。

    • 基本规律（单选 + 辨析）

      规律 条件 行为发生频率 例子
      强化——正强化 愉快刺激 提高行为发生的频率 奖励
      强化——负强化 厌恶刺激 提高行为发生的频率 撤销处分
      惩罚 厌恶刺激 降低行为发生的频率 体罚、谴责
      消退 —- 降低行为发生的频率 不予理睬

    • 教学中的应用（单选 + 辨析）

      • 个人自学、个别化教学
      • 原则：小步子；积极反应；自定步调；及时反馈；低错误率

    • 观察学习又称替代学习、模仿学习，是指通过他人及其强化性结果的观察，获得了某种新的反应，或者矫正了原有的行为反应。
    • 强化的分类（单选）
      • 直接强化：自己做，别人强化你
      • 替代性强化：看别人做，自己被强化

    • 逆向迁移：后对前的影响

  • 迁移的内容

    • 一般迁移：一般原理、方法、策略、态度对另一种学习的影响
    • 具体迁移：具体的、特殊的经验直接迁移到另一种学习中

（三）学习策略#

1. 学习策略的概念（了解）#

学习者在学习活动中，为了达到有效的学习目的而采用的规则、方法、技巧及其调控方式的综合。

2. 学习策略的分类（单选 + 简答）#

  • 认知策略（对外）：认识，知道

    • 复述策略（重复识记）：利用无意识记和有意识记；排除相互干扰；整体识记与分段识记；多种感官参与；复习形式多样化；画线和圈点标注
    • 精加工策略（理解）：记忆术（如位置记忆法、缩减与编歌诀、谐音联想法、关键词法、视觉联想）；做笔记；提问；生成性学习，联系实际生活；利用背景知识
    • 组织策略（整理、归纳）：列提纲；利用图形和表格；归类策略

    例：简述认知策略的内容。

    • 复述策略：复述策略是指为了在记忆中保持所学信息而对信息进行重复识记的策略。
    • 精细加工策略：精加工策略是一种将新学材料与头脑中已有知识联系起来，从而增加新信息的意义的深层加工策略。
    • 组织策略：组织策略是指整合所学新知识之间、新旧知识之间的内在联系，形成新的知识结构的策略。

  • 元认知策略（对内）：对认知过程的认知

    • 计划策略（前）：设置学习目标；浏览学习材料；产生待回答的问题；分析完成任务的方法
    • 监控策略（中）：阅读时对注意加以跟踪；对材料进行自我提问；考试时监视自己的速度和时间（发现问题）
    • 调节策略（后）：根据对认知活动结果的检查，如发现问题，则采取相应的补救措施（解决问题）

  • 资源管理策略

    • 时间管理：统筹安排学习时间；高效利用最佳时间；灵活利用零碎时间
    • 环境管理：调节自然条件（如流通的空气、适宜的温度、明亮的光线以及和谐的色彩等）；设计好学习的空间（如空间范围、室内布置、用具摆放）。
    • 努力管理：激发内在动机；树立学习的信念；选择有挑战性的任务；调节成败标准；正确认识成败的原因；自我奖励
    • 资源利用：学习工具的利用；社会性人力资源的利用（老师的帮助、同学间的合作和探讨等）

  2. 消极的课堂气氛：学生无精打采，采取应付态度，很少主动发言
  3. 对抗的课堂气氛：失控，教师失去了对课堂的控制，学生各行其是

三、班主任工作#

（一）班主任概述（单选）#

班主任是班级的组织者、领导者、教育者，是学校办学思想的贯彻者，是联系班级任课教师和学生团队组织的纽带，是沟通学校、家长和社会的桥梁。

（二）班主任的工作内容（重点）#

1. 班主任的常规工作（单选）#
  • 了解和研究学生：前提和基础
  • 组织和培养班集体：中心环节
  • 建立学生档案
  • 组织班会活动和课外活动
  • 协调各种教育影响
  • 操行评定：自评 - 小组 - 班主任 - 反馈
  • 写好班主任工作计划与总结
2. 个别教育工作（简答 + 材料）#
  1. 优等生
    1. 特点：品学兼优
    2. 教育：严格要求，防止自满；不断激励，提高抗挫折能力消除嫉妒，公平竞争；发挥优势，全班进步
  2. 中等生
    1. 特点：表现平平
    2. 教育：重视对中等生的教育，根据中等生的不同特点有的放矢地进行个别教育。给中等生创造充分展示自己才能的机会，增强他们的自信心
  3. 后进生
    1. 特点：暂时落后
    2. 教育：关心热爱与严格要求结合，培养和激发学习动机，善于发掘“闪光点”，针对差异因材施教，持之以恒

五、教师心理#

（一）教师概述#

1. 教师的含义（单选）#

教师是履行教学职责的专业人员， 承担教书育人（根本任务）、培养社会专业建设人才、提高民族素质的使命。教师职业是一种专门职业， 教师是专业人员。

2. 教师劳动的特点（单选 + 材料）#
  • 复杂性：目的全面、任务多样、对象差异
  • 创造性：因材施教、教学方法、教育机智
  • 连续性
  • 广延性
  • 长期性
  • 间接性
  • 主体性
  • 示范性：其身正，不令而行；桃李不言，下自成蹊
  • 个体性
  • 群体性
3. 教师的领导方式（单选）#
  • 强硬专断型：严加监视，很少予以表扬
  • 仁慈专断型：表扬关心学生，以自己作为班级一切工作的标准
  • 放任自流型：不鼓励，不反对
  • 民主型：合作，平等，鼓励
4. 教师的角色（单选）#
  • 角色构成：多样化
  • 教师的角色意识
    • 角色认知——角色扮演的首要条件
    • 角色体验
    • 角色期待

（二）教师的心理特征（重点：单选）#

  • 教师的认知特征
    • 知识结构
      • 本体性-学科知识
      • 条件性-教育理论
      • 实践性-经验
      • 文化性
    • 教学能力
      • 教学认知-备课
      • 教学设计-写教案
      • 教学操作-实践
      • 教学监控-调节
  • 教师的人格特征
    • 对教学显著影响
      • 教师的热心和同情心
      • 教师富于激励和想象的倾向性
      • 教学效能感：主观判断
  • 教师的行为特征
    • 罗森塔尔效应
      • 内涵：教师的期望或明或暗的传递给学生
      • 别称：皮格马利翁效应、期待效应

（三）教师成长心理（重点）#

1. 教师成长的历程（单选）#
  1. 关注生存：关注教师自身及人际关系
  2. 关注情境：关注学生学习成绩
  3. 关注学生：关注学生的个别差异（成熟的标志）
2. 教师成长与发展的基本途径（单选 + 简答）#
  1. 观摩和分析优秀教师的教学活动 （看别人)
  2. 开展微格教学 (看自己)
  3. 进行专门训练 （多训练）
  4. 反思教学经验 （多反思）
    1. 波斯纳：经验 + 反思 = 成长
    2. 布鲁巴奇：反思日记，详细描述，交流讨论，行动研究
