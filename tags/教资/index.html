教学知识与能力 教育基础知识 一、教育的产生与发展（重点） （一）教育的含义 1. “教育”一词的由来（单选） “教育”一词最早见于**《孟子·尽心上》“**得天下英才而教育之，三乐也” 2. 教育的概念（单选 + 辨析） 广义：泛指一切增进人的知识和技能、发展人的智力和体力、影响人的思想和品德的活动。它包括社会教育、学校教育和家庭教育（三合力）。 狭义：学校教育是教育者根据一定的社会要求，有目的、有计划、有组织地对受教育者施加影响，把他们培养成一定社会所需要的人的活动。 （二）教育的基本要素（单选） 教育者（包括起主体地位的学校教师、管理人员等）：主导作用 教育影响：教育内容、教育手段，纽带、中介 受教育者：学习的主体 （三）教育的属性（重点：单选 + 辨析） 本质属性**（育人性）：教育是有目的培养人的社会活动** 社会属性： 永恒性：人在教育在 教育与人共始终 历史性：古今不同 （历史）继承性：古今相同 长期性：人才培养周期长，如十年树木，百年树人 相对独立性：自身规律、不平衡 教育超前政治、教育落后政治 生产性 （四）教育的功能（单选 + 辨析） 作用对象 个体（本体功能） 个体社会化：自然人 -> 社会人 个体个性化：独特点 社会（派生功能） 社会变迁：推动社会 社会流动：区域 + 职业 性质 正向功能：积极，促进（素质教育） 负向功能：消极，阻碍（应试教育） 呈现形式 显性功能：计划内 隐性功能：计划外 （五）教育的起源 生物起源说：第一个正式学说 观点：教育起源于动物界的生存本能 代表人物：利托尔诺沛西·能 不足： 未说明教育是有目的的 未说明教育是社会活动 主体是动物而非人 心理起源说 观点：教育起源于儿童对成人的“无意识模仿” 代表人物：孟禄 不足： 未说明教育是有目的的 未说明教育是社会活动 劳动起源说：马克思认为其正确 观点：教育起源于劳动过程中社会生产需要和人的发展需要的辩证统一 代表人物：米丁斯基、凯洛夫 （六）教育的发展 1. 原始社会的教育（单选） 无阶级性、公平性 ...

综合素质 职业理念 一、教育观 （一）素质教育的内涵（单选 + 材料分析） 素质教育以提高国民素质为根本宗旨（单选） 包含各方面，贯穿各阶段，大题不写 素质教育不仅适用于基础教育，而且适用于高等教育，还适用于社会教育和终身教育。 提高国民的整体素质，即提高整个中华民族的思想道德素质、科学文化素质、健康素质、审美素质和劳动素质等。 素质教育是面向全体学生的教育（单选 + 材料分析） 关注每一个学生，不抛弃、不放弃 强调在教育中使每个人都得到发展，而不是只注重一部分人，更不是只注重少数人的发展。 素质教育是促进学生全面发展的教育（单选 + 材料分析） 成绩 + 其他 通过德育、智育、体育、美育等的有机融合，来实现学生的德、智、体、美、劳等方面的全面发展。 全面发展不等于平均发展。 素质教育是促进学生个性发展的教育（单选 + 材料分析） 特点 + 特长，因材施教 每个学生都有其独特性，因此，教育还要考虑到学生的个性差异，充分发展学生的个性。 素质教育是以培养学生的创新精神和实践能力为重点的教育（单选 + 材料分析） 新想法、新观点；动手能力，学以致用 （二）新课程改革的教学观 教学从“教育者为中心”转向“学习者为中心” 互动，提问 -> 调动积极性 教师正在从“独奏者”的角色逐渐过渡到“伴奏者”的角色。学生是在教学活动中不应该处于消极被动的地位，而应是积极主动的。 教学从“教会学生知识”转向“教会学生学习” 方法、规律、口诀 今天的教学要立足于教会学生正确思维、正确学习。指导学生掌握学科研究方法，培养学生良好的学习习惯。 教学从“重结论轻过程”转向“重结论的同时更重过程” 不直接给答案，引导学生自己解决。 学生的学习是一个自主发现问题、分析问题、解决问题的过程。 反例：《吉祥三宝》 教学从“关注学科”转向“关注人” 关注全体学生，引导喜怒哀乐，全面发展 关注每一位学生 关注学生的情绪生活和情感体验 关注学生的道德生活和人格养成 （三）答题模板 开头 材料中，教师的行为是正确的。某某老师的教育行为是恰当的，体现了素质教育的理念，值得我们学习。 材料中，教师的行为是错误的。某某老师的教育行为是错误的，违背了素质教育的理念，值得我们反思。 材料中，教师的行为有对有错。某某老师的教育行为有可取之处，也有不当之处，需要我们辩证去看待。 结尾 综上所述，我们在教学中也应该树立正确的素质教育理念，促进学生的发展。 中间 素质教育是面向全体学生的教育。教育中应该使每个学生都得到发展。材料中老师______，体现了老师能够面向全体学生/没有做到一视同仁。 素质教育是促进学生全面发展的教育。教育中不能只关注学生的学习成绩，还要培养学生其他方面的能力。材料中老师______，促进了/不利于学生的全面发展。 素质教育是促进学生个性发展的教育。教育中要考虑到学生的个性差异，发展学生的特点和特长。材料中老师______，促进了/不利于学生个性的发展。 素质教育以培养学生的创新精神和实践能力为重点的教育。教育中教师要支持学生的新想法，培养学生的动手操作能力。材料中老师______，体现了/不利于对学生创新精神和实践能力的培养。 新课改背景下的教育观要求教学从“以教育者为中心”转向“以学习者为中心”。即教师要能够充分调动学生学习的积极性和主动性。材料中老师______，调动了/未调动学生的积极性。 新课改背景下的教育观要求教学从“教会学生知识”转向“教会学生学习”。即要求教师能够教会学生学习的方法，培养学生举一反三的能力。材料中老师______，体现了教师“授人以渔”/没有教会学生学习的方法。 新课改背景下的教育观要求教学从“重结论轻过程”转向“重结论的同时更重过程”。即要求教师能够看到学习过程的重要性，强调学生能够参与学习的过程。材料中老师______，注重了/未注重让学生参与学习的过程。 新课改背景下的教育观要求教学从“关注学科”转向“关注人”。即要求教师能够关注到学生的情绪变化、情感需要，并注重学生人格的养成。材料中老师______，做到了/未做到对学生情绪情感和人格的关注。 二、学生观 （一）“以人为本”的学生观（单选 + 材料分析） 1. 学生是发展的人 学生的身心发展是有规律的（单选） ...
