三、信息隐藏基本原理 3.1 信息隐藏的概念 载体对象：A 打算秘密传递一些信息给 B，A 需要从一个随机消息源中随机选取一个无关紧要的消息 c，当这个消息公开传递时，不会引起怀疑，称这个消息 c 为载体对象。 伪装对象：把需要隐藏的秘密传递的信息 m 隐藏到载体对象 c 中，此时，载体对象 c 就变成伪装对象 c’。 伪装密钥：秘密信息的嵌入过程需要密钥，此密钥称为伪装密钥。 实现信息隐藏的基本要求： 载体对象是正常的，不会引起怀疑 伪装对象与载体对象无法区分，无论从感观上还是从计算机的分析上 不可视通信的安全性取决于第三方有没有能力将载体对象和伪装对象区分开来 对伪装对象的正常处理，不应破坏隐藏的信息 3.2 信息隐藏的分类 无密钥信息隐藏 双方约定嵌入算法和提取算法，算法要求保密 1 2 3 4 5 C：所有可能的载体集合 M：所有可能秘密消息的集合 C’：所有伪装对象的集合 隐藏过程：映射 E —— C * M -> C' 提取过程：映射 D —— C' -> M 无密钥信息伪装系统：对一个五元组 $$\Sigma=<C, M, C’, D, E>$$，其中 C 是所有可能载体的集合，M 是所有可能秘密消息的集合，C’ 是所有可能伪装对象的集合。E：C $$\times$$ M -> C’ 是嵌入函数，D：C’ -> M 是提取函数，若满足对所有 m $$\in$$ M 和 c $$\in$$ C，恒有：D(E(c, m)) = m，则称改五元组为无密钥信息伪装系统。 ...

二、基础知识 2.1 人类听觉特点 2.1.1 语音的产生及数字模型 语言的研究： 从语音的产生和语音的感知方面来研究 从信号处理的角度来研究 人类对语言的认识： 语言的产生：决定想传达给对方的内容是什么，将内容转化成语言的形式（词和语法） 语言学：研究语音中各个音的排列规则及其含义，包括重音、语调、声调等（各国语言，各民族语言，地方方言等） 语音学：研究语音信号时有哪些最基本的单位组成的，发声器官事如何发出声音的，建立语音产生模型，便于人类对语音信号的特性进行研究 语音的产生： 肺部：产生能量 喉部：产生声带的振动 声道（信道）： 口腔 鼻腔 嘴 人能够发出各种不同的声音，取决于声门和声道中各种可变的部分 声带：振动产生浊音，不振动产生清音 声道：一个具有某种谐振特性的腔体，且其横截面积是可变的，声道的频率特性主要取决于声道截面的最小值出现的位置，主要是由舌的位置来控制，嘴是声道的最末端，嘴的形状也影响所发出的声音 用数字模型表示语音信号： 用数字模型分别模拟激励源、声道模型、辐射模型 激励源模型 浊音：激励信号由一个周期脉冲发生器产生，其周期成为基音周期。为了使浊音的激励信号具有声门气流脉冲的实际波形，还需要使这一脉冲序列通过一个声门脉冲模型滤波器，其传输函数为 G(Z) 清音：激励信号由一个随机噪声发生器产生 声道模型：声道可以用一个全极点模型来模拟，其系统函数的极点对应为语音的共振峰 嘴唇辐射模型：嘴唇辐射模型与嘴形有关 语音是时变信号，但可以认为在 10～20 ms 的时间范围内是近似不变的 语音的数字模型是一个缓慢时变的线性系统，在 10～20 ms 的时间内是时不变系统 2.1.2 听觉系统和语音感知 人类说话和收听的过程 影响语音感知的主要因素： 听觉范围 频率范围：正常人可听声音的频率范围为 0.016～16kHz，年轻人可听到 20kHz 的声音，而老年人可听到的最高频率为 10kHz 左右 感觉域：容忍的最高声压。当声压高到一定程度时，耳朵会出现不适感。对正常人而言，一般取 120dB 为不适阈，140dB 为痛阈，且与频率无关 强度差阈：正常人对频率固定对声音所能辨别的最小强度差值 音调：对于频率低的声音听起来感觉它的音调“低”，而频率高的声音听起来感觉它的音调“高”。音调与声音的频率并不成严格的正比关系，它还与声音的强度及波形有关。 掩蔽效应：当人耳听到两个强度不同的声音时，强的声音的频率成分会影响人耳对弱的声音的频率成分的收听 频率掩蔽：低音容易掩蔽高音 时间掩蔽：强声音前面和后面的弱音容易被掩蔽 噪音对单音的掩蔽：一个单音可以被以它为中心频率，具有一定频带宽度的连续噪音所掩蔽 噪音掩蔽的临界带宽：如果在这一频带内噪音的功率等于该纯音的功率，这时该纯音处于刚能被听到的临界状态，则称这一带宽为临界带宽 语音感知和理解：前者在于手机语音信号中所含有的信息，但光靠这些信息还不足以警醒语言理解，还要由收听者根据语法和句法知识对语音信息进行理解。 影响语音清晰度的主要因素： ...

一、概论 1.1 什么是信息隐藏 信息隐藏是一门体现人类高度只会的信息安全斗争技术和艺术 新的信息隐藏手段和技术经常首先被用于情报作战中 在一定程度上决定着战争的胜负乃至国家命运 信息加密给保密信息贴上“此地无银三百两”的标签 数字世界的一个非常重要和紧迫的议题 知识产权保护，如何防止知识产品被非法复制及传播 传统的信息安全技术无法解决上述新问题 信息安全：开发设计不同于传统密码学的技术，即将机密资料信息秘密地隐藏于一般的文件中，然后再通过网络传递 数字水印：用于保护数字作品的知识产权 何谓信息隐藏 所谓信息隐藏的意思，就是将秘密信息秘密地隐藏于另一非机密的文件内容之中，其形式可为任何一种数字媒体，如图像、声音等… 信息隐藏的首要目标是隐藏的技术要好，也就是使加入隐藏信息后的媒体目标的降质尽可能小，达到令人难以察觉的目的 鲁棒性：必须考虑隐藏的信息再经历各种环境、操作之后而免遭破坏的能力 信息隐藏的数据量与隐藏的免疫力始终是一堆矛盾，不存在一种完全满足这两种要求的隐藏方法 信息隐藏技术和密码技术的区别 密码技术：将铭文加密成他人看不懂的密文（乱码）；机密信息不可懂 信息隐藏技术：提供了一种有别于加密的安全模式（调制），其安全性来自于对第三方感知上的麻痹性；机密信息不可见 载体信息提供传递信息的信道，为隐藏信息的传递提供伪装 眼见不一定为实 名词术语： 信息隐藏（Information Hiding）：通过对载体进行难以被感知的改动，从而嵌入信息 隐写术（Steganography）：通过对载体进行难以被感知的改动，从而嵌入秘密信息的技术 数字水印（Digital Watermarking）：通过对载体进行难以被感知的改动，从而嵌入与载体有关的信息，嵌入的信息不一定是秘密地，也有可能是可见的 隐写分析（Steganalysis）：检测、提取、破坏隐写对象中秘密信息的技术 根据载体的特征选择合适的信息隐藏算法： 信息隐藏的载体：图像、音频、视频、网络协议、文本和各类数据等 有冗余：图像、视频、音频中的信息隐藏，大部分是利用了人类感官对于这些载体信号的冗余来隐藏信息 无冗余：文本、网络协议和各类数据等就无法利用冗余度来隐藏信息，因此在这些没有冗余度或冗余度很小的载体中隐藏信息，需要采用其他方法 信息隐藏的两个重要研究分支： 隐写术 数字水印 保护对象 保护嵌入到载体中的秘密信息 隐藏信息的载体 不可察觉性 √ - 不可检测性 √ - 隐藏容量 √ - 鲁棒性 - √（脆弱水印除外） 不可见 √ -（也有可见水印） 1.2 信息隐藏的历史回顾 1.2.1 技术性的隐写术 文头送信的方法 书记板的方法 实物隐藏法 标记法 微缩胶片法：缩拍相片 化学方法的隐写术 艺术作品中的隐写术 回声法 嵌入隐匿标记：利用掩蔽材料的预定位置上某些误差和风格特性来隐藏信息 利用字的标准体和斜体来编码 将版权信息和序列号隐藏在行间距和文档的其他格式特性之中 通过对文档的各行提升或降低来表示 0 或 1 等 1.2.2 语言学中的隐写术 诗情画意传“密语”：藏头诗，藏尾诗等 首（次）字母法 卡登格子法 乐谱法 1.2.3 保护版权的隐写术 核对校验图（作品的摘要） 纸张中的水印 纸币中的水印 1.3 发展现状和分类 1.3.1 信息隐藏技术的发展现状 20 世纪 90 年代开始迅速发展 ...
