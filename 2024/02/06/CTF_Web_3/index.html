    • 对于 ssh 服务的 22 端口的靶场

      • 暴力破解

    • 对于开放 http 服务的 80 端口或其他端口的靶场

      • 对于开放的 http 服务的大端口，可以采用 http://ip:port/ 的形式访问

      • 查看源代码获取对应信息，如果没有 flag 信息，使用工具探测隐藏页面

文件上传漏洞 & 一句话木马

文件上传流程：

  1. Client（浏览器）文件表单提交
  2. Server：生成临时文件
  3. 服务器端代码处理，如 PHP：读取临时文件信息，判断文件是否合规，保存文件

最常见的利用文件上传漏洞的方法就是上传网站木马（webshell）文件。WEBSHELL 又称网页木马文件，氛围 ASP 木马、PHP 木马、JSP 木马等，利用脚本语言中的系统命令执行、文件读写等函数的功能，一旦上传到服务器被脚本引擎解析，攻击者就可以实现对服务器的控制。
