信息隐藏技术_1

一、概论

1.1 什么是信息隐藏

信息隐藏是一门体现人类高度只会的信息安全斗争技术和艺术

  • 新的信息隐藏手段和技术经常首先被用于情报作战中
  • 在一定程度上决定着战争的胜负乃至国家命运
  1. 信息加密给保密信息贴上“此地无银三百两”的标签

  2. 数字世界的一个非常重要和紧迫的议题

    1. 知识产权保护，如何防止知识产品被非法复制及传播

  3. 传统的信息安全技术无法解决上述新问题

    1. 信息安全：开发设计不同于传统密码学的技术，即将机密资料信息秘密地隐藏于一般的文件中，然后再通过网络传递
    2. 数字水印：用于保护数字作品的知识产权

  4. 何谓信息隐藏

    1. 所谓信息隐藏的意思，就是将秘密信息秘密地隐藏于另一非机密的文件内容之中，其形式可为任何一种数字媒体，如图像、声音等…
      1. 信息隐藏的首要目标是隐藏的技术要好，也就是使加入隐藏信息后的媒体目标的降质尽可能小，达到令人难以察觉的目的
      2. 鲁棒性：必须考虑隐藏的信息再经历各种环境、操作之后而免遭破坏的能力
      3. 信息隐藏的数据量与隐藏的免疫力始终是一堆矛盾，不存在一种完全满足这两种要求的隐藏方法

  5. 信息隐藏技术和密码技术的区别

    1. 密码技术：将铭文加密成他人看不懂的密文（乱码）；机密信息不可懂
    2. 信息隐藏技术：提供了一种有别于加密的安全模式（调制），其安全性来自于对第三方感知上的麻痹性；机密信息不可见
      1. 载体信息提供传递信息的信道，为隐藏信息的传递提供伪装

  6. 眼见不一定为实

名词术语：

  1. 信息隐藏（Information Hiding）：通过对载体进行难以被感知的改动，从而嵌入信息

  2. 隐写术（Steganography）：通过对载体进行难以被感知的改动，从而嵌入秘密信息的技术

  3. 数字水印（Digital Watermarking）：通过对载体进行难以被感知的改动，从而嵌入与载体有关的信息，嵌入的信息不一定是秘密地，也有可能是可见的

  4. 隐写分析（Steganalysis）：检测、提取、破坏隐写对象中秘密信息的技术

根据载体的特征选择合适的信息隐藏算法：

  1. 信息隐藏的载体：图像、音频、视频、网络协议、文本和各类数据等
  2. 有冗余：图像、视频、音频中的信息隐藏，大部分是利用了人类感官对于这些载体信号的冗余来隐藏信息
  3. 无冗余：文本、网络协议和各类数据等就无法利用冗余度来隐藏信息，因此在这些没有冗余度或冗余度很小的载体中隐藏信息，需要采用其他方法

信息隐藏的两个重要研究分支：

隐写术 数字水印
保护对象 保护嵌入到载体中的秘密信息 隐藏信息的载体
不可察觉性 √ -
不可检测性 √ -
隐藏容量 √ -
鲁棒性 - √（脆弱水印除外）
不可见 √ -（也有可见水印）

1.2 信息隐藏的历史回顾

1.2.1 技术性的隐写术

  1. 文头送信的方法

  2. 书记板的方法

  3. 实物隐藏法
  4. 标记法
  5. 微缩胶片法：缩拍相片
  6. 化学方法的隐写术
  7. 艺术作品中的隐写术
  8. 回声法
  9. 嵌入隐匿标记：利用掩蔽材料的预定位置上某些误差和风格特性来隐藏信息
    1. 利用字的标准体和斜体来编码
    2. 将版权信息和序列号隐藏在行间距和文档的其他格式特性之中
    3. 通过对文档的各行提升或降低来表示 0 或 1 等

1.2.2 语言学中的隐写术

  1. 诗情画意传“密语”：藏头诗，藏尾诗等
  2. 首（次）字母法
  3. 卡登格子法
  4. 乐谱法

1.2.3 保护版权的隐写术

  1. 核对校验图（作品的摘要）
  2. 纸张中的水印
  3. 纸币中的水印

1.3 发展现状和分类

1.3.1 信息隐藏技术的发展现状

  1. 20 世纪 90 年代开始迅速发展

    1. 1992 年国际上正式提出信息隐形研究
    2. 1996 年国际第一届信息隐藏研讨会（IHW）
    3. 1999 年中国第一届信息隐藏研讨会（CIHW）

  2. 90 年代，多个组织开始考虑将数字水印技术纳入标准

  3. 90 年代后期，一些公司推出了水印产品
  4. 理论研究的热潮（2000 年以后）
    1. 隐写术：伪装式保密通信
    2. 数字水印：数字产品版权保护
    3. 信息分存：可视密码
    4. 隐蔽信道：计算机系统中的一些通道
    5. 数字图像取证：真实性鉴别

1.3.2 伪装式保密通信

伪装式保密通信是古典隐写术与现代技术的直接结合。

  • 隐藏载体：利用多媒体信息作为隐藏载体
  • 可视密码学：把要隐藏的密钥信息通过算法隐藏到两个或多个子密钥图片中，每一张图片上都有随机分布的黑点和白点，把所有的图片叠加在一起，则能回复出原有的信息。
  • 叠像术

1.3.3 数字水印

  1. 知识产权的定义和内容
  2. 数字产品的版权保护
  3. 标记手段：
    1. 数字水印：嵌入在数字作品中的一个版权信息，可以给出作品的作者、所有者、发行者以及授权使用者等版权信息
    2. 数字指纹：作为数字作品的序列码，用于跟踪盗版者

1.3.4 总结

  • 信息隐藏技术的最重要的两个分支是数字水印、隐写术
  • 数字水印主要用于版权保护
  • 隐写术主要用于保密通信
  • 数字水印嵌入载体中的信息与载体有关
  • 隐写术则前途与载体无关的秘密信息
  • 基于隐写术的保密通信掩盖的是通信存在的事实

1.4 信息隐藏算法的性能指标

  1. 透明性（首要特性）：嵌入的秘密信息导致隐写载体信号质量的变化程度。
  2. 鲁棒性（稳健性）：指隐藏的秘密信息抵抗各种信号处理和攻击的能力。鲁棒性水印通常不会因常见的信号处理和攻击而丢失隐藏的水印信息。
  3. 隐藏容量：指单位时间内或一幅作品中能嵌入水印的比特数。

三个性能指标之间相互制约，没有一种算法能让这三个指标达到最优

  1. 透明性较好时，鲁棒性较差
  2. 鲁棒性较好时，透明性较差
  3. 信息容量较大时，透明性较差

1.5 可视密码学

可视化密码是一种依靠人眼解密的秘密共享方法：将一个秘密图像加密成 n 张分享图像，由 n 个人分比如保管，解密时只需要 k’ (k‘ >= k) 个人将各自的分享图像叠加，秘密图像就会呈现，而少于 k 个人无法获得秘密图像的任何一点信息。

  1. Shamir 的二值分存方案
  2. 灰度图像的可视密码技术实现：误差扩散法
  3. 信息分存
  4. 彩色图像可是密码技术实现
